###  Что такое шаблоны ООП

**Шаблоны ООП** — это принципы и структуры, позволяющие проектировать гибкий и расширяемый код.  
В Java они реализуются через:
- **Интерфейсы** — задают общие методы для разных классов (`Fractionable`, `Meowable`);
- **Инкапсуляцию** — скрывают внутреннее состояние объектов;
- **Полиморфизм** — единый интерфейс для разных реализаций;
- **Переопределение методов** (`toString`, `equals`, `hashCode`);
- **Generics (Дженерики)** — обобщённые классы и методы, работающие с любыми типами.

###  Что такое коллекции

**Коллекции** — это стандартные структуры Java для хранения данных:
- `List` — список с индексами;
- `Set` — множество без повторов;
- `Queue` — очередь (FIFO);
- `Map` — ассоциативный массив (ключ–значение).

Коллекции упрощают работу с наборами данных — добавление, удаление, поиск, сортировку и фильтрацию.

### **Задание 1. Класс «Дробь»**

**Цель:** Реализовать интерфейс `Fractionable` и класс `Fraction`,  
который хранит числитель и знаменатель, вычисляет значение и сравнивает дроби.

**Основные методы:**
- `toDouble()` — переводит дробь в вещественное значение;  
- `equals()` — сравнивает дроби по числителю и знаменателю;  
- Кэширование значения для оптимизации.

**Пример работы:**
```
Введите числитель: 1
Введите знаменатель: 2
Дробь: 1/2 = 0.5
Новый числитель: 3
После изменения: 3/2 = 1.5
Сравнение с другой дробью 3/2: true
```

### **Задание 4. Олимпиада**

**Цель:** Считать из файла список участников и вывести тех, кто набрал максимальное количество баллов.

**Пример входного файла (`input.txt`):**
```
3
Петрова Ольга 25 18 16
Калиниченко Иван 14 19 15
Морозов Сергей 25 18 16
```

**Пример вывода:**
```
Максимальный результат: 59
Победители:
Петрова Ольга
Морозов Сергей
```
